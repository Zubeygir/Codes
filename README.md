Proje Hakkında

Bu proje, büyük boyutlu şifre listelerini işlemek, hashlemek, indekslemek ve hızlı şekilde arama yapmak amacıyla geliştirilmiş bir Node.js uygulamasıdır.

Sistem;

Şifreleri satır satır okur
MD5, SHA1 ve SHA256 hashlerini üretir
İlk karaktere göre indeksler
Dosya bazlı saklama yapar
Arama işlemlerini optimize eder
İşlenen dosyaları farklı klasöre taşır

Bu proje bir ödev kapsamında geliştirilmiştir.


Kullanılan Teknolojiler
Node.js
File System (fs)
Path
Crypto
Readline
